# 🏝️ Tripcore — Pitch Deck ala Sosmed

> Dibikin iseng karena males catet trip di grup WA. Sekarang udah live di [tripcore-beta.vercel.app](https://tripcore-beta.vercel.app/).

---

## Slide 1 — The Hook

```
A: "Bro itinerary day 2 gimana?"
B: "Cek grup WA, scroll ke atas"
A: "Yang mana? Udah ketimbun 500 chat"
B: "Yaudah deh gue catet ulang"

😭😭😭 RELATABLE BANGET GAK SIH?
```

**Masalah:** Trip planning di grup WhatsApp = chaos. Itinerary ilang ketimbun chat, duit kas susah dilacak, patungan bingung siapa udah bayar siapa belum.


## Slide 6 — Tech Flex (buat yang kepo)

```
⚛️ React 19 + TypeScript
🎨 Tailwind v4 + Motion animation
🔥 Firebase Firestore real-time sync
📱 PWA — installable, works offline

```

Dibangun weekend-iseng. Tapi proper engineering:
- Offline-first (IndexedDB persistence)
- Anonymous auth + PIN identity

- Auto debt settlement algorithm

## Slide 7 — Use Case

```
🌴 Weekend getaway ke Bali
   10 orang, 3 mobil, 1 villa

Sebelum Tripcore:
- Catet budget di Notes → ilang
- Split bill di kalkulator → ribet
- Itinerary di Google Docs → gak ada yang baca
- Checklist di grup WA → "eh tadi siapa yang bawa sunblock?"

Sesudah Tripcore:
- Buka app → semua ada
- Bayar patungan → auto ke-track
- Itinerary → clear, day-by-day
- Checklist → real-time, semua tau progres
- Pulang → gak ada drama "duit gue kurang nih"
```
